# 视频转 SRT 字幕工具 - 规格说明书

## 1. 项目概述

| 项目名称 | Video to SRT |
|----------|--------------|
| 类型 | 桌面工具 (本地运行) |
| 核心功能 | 将长视频(3-4小时)批量转换为 SRT 中文字幕 |
| 目标用户 | 需要从视频中提取字幕供大模型分析的用户 |

## 2. 技术栈

| 组件 | 选择 | 说明 |
|------|------|------|
| 语言 | Python 3.11+ | |
| 界面 | Streamlit | 快速开发，支持文件上传 |
| STT 引擎 | whisper.cpp | M4 CoreML 加速，本地运行 |
| 音频处理 | FFmpeg | 视频转音频 |
| 字幕格式 | SRT | 唯一输出格式 |

### 依赖

```
streamlit>=1.28.0
whispercpp-python>=0.1.0
ffmpeg-python>=0.2.0
```

| 功能 | 描述 | 优先级 |

| 模型 | 参数量 | 内存 | 速度 | 推荐场景 |
|------|--------|------|------|----------|
| tiny | 39M | ~1GB | 最快 | 快速测试 |

## 5. 处理流程

### 5.1 主流程

```
用户上传视频
    │
    ▼
检查文件格式 (mp4, avi, mkv, mov, webm)
    │
    ▼
FFmpeg 提取音频 → temp/audio.wav
    │
    ▼
Whisper CoreML 识别 → 生成 SRT
    │
    ▼
检查 SRT 大小
    │
    ├── < 10MB → 直接保存
    │
    └── ≥ 10MB → 按时间切分 (每段30分钟)
                │
                ▼
            保存多个 SRT 文件
                │
                ▼
清理临时文件
```

### 5.2 分段策略

| 原始视频时长 | Whisper 分段 | 最终输出 |
|--------------|--------------|----------|
| ≤ 30分钟 | 1段 | 1个 SRT |
| 30-60分钟 | 2段 | 1个 SRT (合并) |
| 60-90分钟 | 3段 | 1个 SRT (合并) |
| 3-4小时 | 6-8段 | 1个 SRT (合并) |

### 5.3 SRT 切分策略 (输出)

| SRT 大小 | 切分方式 |
|----------|----------|
| < 10MB | 单文件 |
| ≥ 10MB | 每 10MB 或每 30分钟一段 |

## 6. 数据格式

### 6.1 SRT 格式示例

```srt
1
00:00:00,000 --> 00:00:02,500
这是第一句字幕

2
00:00:02,600 --> 00:00:05,300
这是第二句字幕

3
00:00:05,400 --> 00:00:08,100
这是第三句字幕
```

### 6.2 切分后的文件名

```
原文件: video.mp4
输出:
  ├── video_part1.srt
  ├── video_part2.srt
  └── video_part3.srt
```

## 7. 错误处理

| 错误类型 | 处理方式 | 用户提示 |
|----------|----------|----------|
| 文件格式不支持 | 拒绝上传 | "不支持该格式，请上传 mp4/avi/mkv/mov/webm" |
| 文件过大 (>10GB) | 拒绝上传 | "文件过大，请上传小于 10GB 的视频" |
| FFmpeg 失败 | 重试2次 | "音频提取失败，请检查视频文件完整性" |
| Whisper 失败 | 重试2次 | "语音识别失败，请重试" |
| 磁盘空间不足 | 检查并提示 | "磁盘空间不足，请清理后重试" |
| 输出目录不存在 | 自动创建 | (静默处理) |
